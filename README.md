# Lucca Menu

هذا المشروع هو نسخة منيو مطعم Lucca قابلة للتعديل بسهولة.

## كيف تستخدمه

1. افتح `index.html` في المتصفح.
2. لتعديل الأصناف أو الأسعار، افتح `menu-data.js` في محرر نصوص.
3. احفظ التغييرات ثم حدث الصفحة في المتصفح.

## كيف تجعل المنيو متاح 24/7

### خيار 1: GitHub Pages
- أنشئ مستودع على GitHub.
- ضع ملفات المشروع في المستودع (`index.html`, `styles.css`, `script.js`, `menu-data.js`).
- فعّل GitHub Pages من إعدادات المستودع واختر الفرع `main` أو `master` من المجلد `/`.
- سيكون الرابط متاحاً دائماً عبر الإنترنت.

### خيار 2: استضافة محلية على هذا الجهاز
- إذا كان جهازك يعمل دائماً، يمكنك تشغيل سيرفر محلي.
- إذا كان Node.js مثبتاً، افتح PowerShell في مجلد `Lucca-menu` وشغل:
  ```powershell
  npx http-server .
  ```
- افتح الرابط `http://127.0.0.1:8080` في المتصفح.

## كيفية تعديل المنيو

في `menu-data.js` يمكنك:
- إضافة فئة جديدة `title` و `icon` و `items`
- تعديل `name` و `price` و `unit`
- إضافة وصف بـ `description`
- إضافة أسعار متعددة بـ `prices`
- إضافة اختيار مصدر للقهوة بـ `origins`

## ملاحظات

- لو رفعتها على GitHub Pages تصبح متاحة دائماً عبر الإنترنت.
